⛰️ Maastopyörä

⛰️ Maastopyörä

Maastopyörä on suunniteltu epätasaisille reiteille kuten metsään ja vuoristoon.

  • ✔ Leveät renkaat
  • ✔ Iskunvaimennus
  • ✔ Hyvä pito vaikeissa maastoissa
  • ✔ Kestävä runko
  • ❌Raskas ajaa pitkillä matkoilla asfaltilla
  • ❌Vaatii enemmän voimaa ylämäissä

🏁 Maantiepyörä

Nopea ja kevyt asfalttiajoon.

🏁 Maantiepyörä

Maantiepyörä on suunniteltu nopeaan ajoon kovalla alustalla.

  • ✔ Kevyt runko
  • ✔ Kapeat renkaat
  • ✔ Aerodynaaminen muotoilu
  • ✔ Sopii pitkille matkoille
  • ❌Ei sovi huonoille tai epätasaisille teille
  • ❌Erittäin ohut rengas voi puhjeta helpommin

🤸 BMX

Temppuihin ja skeittiparkkeihin.

🤸 BMX

BMX on pieni ja kestävä pyörä temppuihin ja freestyle-ajoon.

  • ✔ Pieni runko
  • ✔ Erittäin kestävä
  • ✔ Sopii temppuihin
  • ✔ Ei vaihteita
  • ❌Ei vaihteita → raskas ylämäissä
  • ❌Rajoitettu käyttö vain temppuihin ja puistoihin

🪨 Gravel

Monipuolinen tie + sora + metsä.

🪨 Gravel

Gravel-pyörä yhdistää maantie- ja maastopyörän ominaisuudet.

  • ✔ Monipuolinen käyttö
  • ✔ Sopii sorateille
  • ✔ Mukava pidemmillä matkoilla
  • ✔ Vakaa ajo
  • ❌Ei yhtä kestävä kuin maastopyörä äärireiteillä
  • ❌Voi olla kallis verrattuna peruspyöriin
